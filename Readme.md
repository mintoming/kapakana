# カパかな

## フォント概要

2018年にリリースしたフォント「かぱかな」の更新版です。

カッパープレート体をモチーフにしたひらがな書体を作りました。

2020年10月を目標に下記内容の更新を行います。

・カタカナの追加

・バリアブルフォント化

・Pinyon Script体とのスタイル統一
